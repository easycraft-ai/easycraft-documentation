10.1 General Configuration

General Configuration Items

General configuration items are system-wide settings, mainly including:

  • Portal search address configuration
  • Personal settings page-related configurations
  • System default address configuration
  • Intranet address configuration
  • System switch configurations

Parameter Description​

Parameter NameConfiguration Description
Search PC/Mobile AddressConfiguration of search address in the portal header.
- EasyCraft Smart Search Address: http://111.119.250.44/web/labc-search/#/?q=!{keyword}
- EasyCraft Full-Text Search Address: http://111.119.250.44/web/labc-search/#/?q=!{keyword}
Personal Information PageConfiguration of related page addresses in personal settings.
- This System: Built-in pages of this system.
- Third-Party System: Supports extending third-party system addresses.
Password Recovery PageConfiguration for Forgot Password on the login page.
- Disabled: Login page has no "Forgot Password" feature.
- This System: Link points to this system’s recovery page.
- Third-Party System: Link points to a third-party recovery page.
Modify Password PageConfiguration of the change password address in personal settings.
- This System: Uses the built-in password modification page.
- Third-Party System: Supports linking to third-party system password pages.
Image DisplayConfiguration for EasyCraft user avatar addresses.
- This System: Uses default avatar from the organizational structure.
- Third-Party System: Uses avatar from third-party systems.
Intranet Backend Management DomainSupports intranet/extranet domain access.
⚠️ The intranet domain must be accessible; otherwise, EasyCraft cannot be accessed internally.
Default Homepage PC/Mobile AddressDefines the default homepage address when the EasyCraft system does not have a portal management module.
- If empty, the system provides a default homepage.
Favorites ToggleControls the portal favorites function.
- Show: Users can enable/disable favorites in personal settings.
- Hide: Favorites toggle is hidden; the system disables favorites and user customization.
Browser IconBrowser favicon for the EasyCraft system.
- Recommended size: 32×32
- Format: ICO only
System LogoRecommended image height: 32px

Notes​

  • !{keyword} in search addresses is a placeholder for dynamic search terms.
  • Ensure intranet domain configuration is correct for proper system accessibility.
  • Only ICO format is supported for browser icons.
  • The Favorites toggle is critical for user customization of portal features.
